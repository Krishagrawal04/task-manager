# Task Manager

A simple task manager web app built with HTML, CSS, and JavaScript. Tasks are saved to localStorage so they persist across browser sessions.

## Features

- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Clear all completed tasks
- Data persists in localStorage

## Quick Deploy

Run the following command to fully deploy the app to GitHub Pages:

```bash
bash setup.sh
```

This script will:
1. Initialize a git repository
2. Create a public GitHub repository
3. Push code to GitHub
4. Enable GitHub Pages automatically

After deployment, your app will be live at:
`https://yourusername.github.io/task-manager/`

## Local Development

Open `index.html` in any web browser to use the app locally.

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub Pages for hosting
